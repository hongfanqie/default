# default

Github Pages default theme.

Variables:

```
site.url: '{{ site.url }}'
site.url | absolute_url: '{{ site.url | absolute_url }}'
site.baseurl: '{{ site.baseurl }}'
site.baseurl | absolute_url: '{{ site.baseurl | absolute_url }}'
"/" | absolute_url: {{ "/" | absolute_url }}
"/" | relative_url: {{ "/" | relative_url }}
```
